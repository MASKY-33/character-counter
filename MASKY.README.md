# Character Counter
This program asks the user for a sentence, counts how many characters it contains (including spaces), and prints the total.

## Features
- Asks the user to type a sentence
- Counts all characters using len()
- Includes spaces and punctuation in the count
- Prints the total number of characters


## Learning Purpose
- A small exercise to practice:
- user input
- string length (len())
- understanding characters vs. words
